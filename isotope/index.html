Layout Mode

  • Built in layout modes: masonry (default), fitRows, vertical

Masonry Mode

  • Horizontal order is not guaranteed to work
  • If columnWidth option is not set, it's set the first item's width. If it is set, it has to be set to a fixed number representing number of pixels. Percentage cannot be set
