  • touchstart and touchmove listeners by default have passive:true in AddEventListenerOptions which means preventDefault can't be used and if so it will throw error [Intervention] Unable to preventDefault inside passive event listener due to target being treated as passive. Add .owl-carousel {touch-action:manipulation;} to prevent the error.

CSS class prefix owl-*

  • .owl-carousel.owl-theme .item
  • .item img by default is 100% width
  • Owl modifies the original HTML
  • Layout is re-calculated when viewport is changed
  • Add padding for all .item: .owl-carousel.owl-theme.lili-ex .item {padding:...}

Loop Dynamic Width with Dynamic Height and Margin

  • By default width is equal based on number of .item.
  • This sets the .item width to auto $('#lili-ex-4').owlCarousel({loop:true,autoWidth:true, margin:10, items:3 });
  • However, option:items:4, option:loop:true won't have effect, the number of items depend on the wrapper's total width.
  • Height is not fixed and it's set to the biggest calculated height of an item.
