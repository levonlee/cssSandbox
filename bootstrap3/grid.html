Center one column

  • .col-*.col-*-offset-[n] where n = (12 - number of columns) / 2
  • Only works even number of columns
  • .col-*-offset-n is margin-left: (n/12)*100%

  • So .col-md-offset-4
  • Set .col-md-offset-4 only for the first column
