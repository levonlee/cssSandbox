Example 1

  • nav.navbar.navbar-expand{-sm|-md|-lg|-xl}.navbar-dark
    • Always collapse nav.navbar.navbar-expand
    • Placement nav.navbar.fixed-top.navbar-expand{-sm|-md|-lg|-xl}.navbar-dark
      • .fixed-top
      • .fixed-bottom
      • .sticky-top (position:sticky not supported in IE11)
    • Align navbar components nav.navbar.justify-content-center
    • .navbar-light .navbar-dark .navbar-light.bg-*
    • Force navbar to be printed nav.navbar.d-print
    • If not nav, then div[role="navigation"]
  • Direct children of nav.navbar could be:
    • a.navbar-brand
      • Align left a.navbar-brand.mr-auto
    • button.navbar-toggler[data-toggle="collapse" data-target="#navbarSupportedContent" aria-controls="navbarSupportedContent" aria-expanded="false" aria-label="Toggle navigation"]
      • If there's no element before .navbar-toggler, then toggler is aligned to left
      • If there's element, then toggler is aligned to right
    • div#navbarSupportedContent.collapse.navbar-collapse
      • Any content can be put here but usually ul.navbar-nav. Not ul.nav!
      • ul.navbar-nav li.nav-item a.nav-link
        • Align right ul.navbar-nav.ml-auto.justify-content-end
        • li.nav-item.active
  • Use div.container to wrap navbar components e.g. div.container a.navbrand

Nav

Basics

  • div[role=navigation] ul.nav li.nav-item a.nav-link.active
  • When non ul tag is used, the parent of the ul should have attribute [role=navigation]
  • Can be nav.nav. .nav uses display:flex
  • Dropdown on items li.nav-item.dropdown
  • To center items ul.nav.justify-content-center
  • Vertical ul.nav.flex-column
  • Tabs ul.nav.nav-tabs
  • Pills ul.nav.nav-pills
  • Fill and Justify ul.nav.nav-pills.nav-fill
  • Items with equal width ul.nav.nav-pills.nav-justified
  • Small vw or above use row, below use column ul.nav.nav-pills.flex-column.flex-sm-row
