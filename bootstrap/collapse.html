Multiple targets

  • Use href or data-target in buttons and links to refer to multiple elements
  • First link uses href only
  • Second element, button, uses data-target only
  • Third element, button, uses data-target=".multi-collapse" to refer to target CSS classes
  • Buttons and links use aria-controls="" to refer to the target element id. If multiple, separate them by space
  • Use role="button" for a
  • Use aria-expanded true or false to show the current state of the collapsible element
  • When aria-expanded="true", make sure collapsible element has show class.
  • Add data-toggle="collapse" and a data-target to the element to automatically assign control of one or more collapsible elements. The data-target attribute accepts a CSS selector to apply the collapse to.
  • Be sure to add the class collapse to the collapsible element. If youâ€™d like it to default open, add the additional class show.
